# Vorlage - Brief

Die Vorlage zeigt, wie man mit LaTeX und der Dokumentenklasse ``scrlttr2`` einen Brief erstellt.

Mit etwas LaTeX-Erfahrung sollte man die Vorlage einfach an die eigenen Bedürfnisse anpassen können.
